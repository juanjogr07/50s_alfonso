Como subir a GitHub Pages
Pasos para publicar las invitaciones en internet gratis.
  1. Crea una cuenta en github.com si no tienes una
  2. Crea un nuevo repositorio (puede ser privado o publico)
  3. Sube los archivos: invitation.html y el folder completo
  4. Ve a Settings → Pages → Source: Deploy from branch → main
  5. Tu URL base sera: https://TU-USUARIO.github.io/REPOSITORIO
  6. Cada invitado recibe: https://TU-USUARIO.github.io/REPOSITORIO/invitation.html?guest=nombre-apellido
